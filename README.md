##### windows 下使用方法
```
-1 如果你没有安装过python
	去 https://www.python.org/downloads/  
	下载并且安装 python 2.7  
-2
	从我这里下载 main.py

-3 以 <strong> 管理员 </strong> 身份运行cmd(命令指示符)
	运行以下命令
	python 下载的main.py的路径

	例如
	python C:\Downloads\main.py
```

#####.nix 下使用方法
```
sudo python 下载的main.py的路径
```
